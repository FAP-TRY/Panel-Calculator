# Kalkulator Panel Tritunggal Swarna

Aplikasi desktop Windows untuk kalkulasi harga panel listrik PT Tritunggal Swarna.

## Fitur

- Katalog produk 8.000+ item (Schneider, Himel, HOWIG)
- Kalkulator estimasi dengan margin bertingkat, PPN/PPh, ongkir
- Export PDF (Surat Formal & Modern) dan CSV
- Riwayat estimasi dengan filter dan load ulang
- Auto-update via GitHub Releases
- Manajemen pengguna (Admin/Operator)
- Import produk dari CSV/Excel

## Versi Terkini

**v1.2.4** — lihat [Releases](https://github.com/FAP-TRY/Panel-Calculator/releases)

## Instalasi

Download installer dari halaman Releases:
`KalkulatorPanel-TTS-vX.Y.Z-Setup.exe` — password: **TTS2025**

## Tech Stack

- C# .NET 8 · WinForms · SQLite · Entity Framework Core · iText7
